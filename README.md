Chạy lệnh sau để cài toàn bộ thư viện cần thiết:
pip install -r requirements.txt

Lưu ý:
Do các file .csv của bộ dữ liệu CICIDS2017 khá lớn nên GitHub không hỗ trợ push trực tiếp toàn bộ dataset lên repository.
Vì vậy, khi pull project về máy, mọi người cần tự thêm dataset vào theo README của Phú.
